Progressive Web App は新しい技術ではなく今ままでの技術を組み合わせたもので、ただアプリをつくるのではなくよりネイティブアプリっぽい快適な動作をするウェブアプリケーションを Progressive Web App と呼ぶ。

* https://developers.google.com/web/progressive-web-apps/
