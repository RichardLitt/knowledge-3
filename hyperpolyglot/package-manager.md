# Package Manager Hyperpolyglot

|     | [npm] | [bundler] |
| --- | --- | --- | --- |
| Create configuration file | [`npm init`][npm-init] | [`bundle init`][bundler-init] |
| Install a package | [`npm i || install`][npm-install] | [`bundle install --path vendor/bundle`][bundler-install] |
